## Les algorithmes incontournables du Machine Learning

Chaque algorithme a ses forces, ses faiblesses et ses cas d'usage. Voici les plus importants à connaître.

## Algorithmes de classification

### Régression logistique

Malgré son nom, la **régression logistique** est utilisée pour la **classification**. Elle prédit la probabilité qu'une observation appartienne à une classe.

**Quand l'utiliser :**
- Classification binaire (oui/non, spam/pas spam)
- Quand vous avez besoin d'interprétabilité
- Comme baseline de référence

### Arbres de décision

Les **arbres de décision** divisent les données en suivant une série de questions (conditions). Chaque nœud représente une question, chaque branche une réponse.

```
Le patient a-t-il de la fièvre ?
├── Oui → A-t-il de la toux ?
│   ├── Oui → Grippe probable
│   └── Non → Infection possible
└── Non → Pas de grippe
```

**Avantages :** Faciles à comprendre et à visualiser
**Inconvénients :** Tendance à l'overfitting

### Random Forest

Le **Random Forest** combine plusieurs arbres de décision pour obtenir un résultat plus robuste. C'est le principe de la **sagesse des foules** appliqué au ML.

**Fonctionnement :**
1. Créer N arbres de décision sur des sous-ensembles aléatoires
2. Chaque arbre vote
3. La majorité l'emporte

### SVM (Support Vector Machine)

Les **SVM** cherchent l'hyperplan qui sépare le mieux deux classes de données. Ils sont particulièrement efficaces en haute dimension.

## Algorithmes de régression

### Régression linéaire

La **régression linéaire** cherche la droite qui s'ajuste le mieux aux données pour prédire une valeur continue.

```
prix_maison = a * surface + b * nb_chambres + c * localisation + d
```

**Cas d'usage :** Prédiction de prix, estimation de ventes, prévisions

### Régression polynomiale

Quand la relation n'est pas linéaire, on utilise des **polynômes** pour capturer des relations plus complexes.

## Algorithmes de clustering

### K-Means

**K-Means** regroupe les données en K clusters en minimisant la distance entre chaque point et le centre de son cluster.

**Étapes :**
1. Choisir K centres aléatoires
2. Assigner chaque point au centre le plus proche
3. Recalculer les centres
4. Répéter jusqu'à convergence

**Applications :** Segmentation de marché, compression d'images, détection d'anomalies

### DBSCAN

**DBSCAN** (Density-Based Spatial Clustering) découvre des clusters de forme arbitraire en se basant sur la densité des points. Il détecte aussi les points aberrants.

## Comment choisir son algorithme ?

| Situation | Algorithme recommandé |
|-----------|----------------------|
| Classification simple | Régression logistique |
| Classification complexe | Random Forest ou Gradient Boosting |
| Prédiction de valeur | Régression linéaire |
| Grouper des données | K-Means |
| Données en haute dimension | SVM |
| Première approche | Random Forest (bon par défaut) |

## Points clés à retenir

- Il n'existe pas d'algorithme universel — le choix dépend du problème
- **Random Forest** est souvent un bon point de départ
- La **régression logistique** n'est pas une régression mais une classification
- Le **clustering** (K-Means, DBSCAN) est utilisé pour l'apprentissage non supervisé
