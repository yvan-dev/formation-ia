## Qu'est-ce qu'un réseau de neurones ?

Un **réseau de neurones artificiel** est un modèle de machine learning inspiré du fonctionnement du cerveau humain. Il est composé de couches de "neurones" interconnectés qui traitent l'information.

## Le neurone artificiel

Un **neurone artificiel** (ou perceptron) est l'unité de base d'un réseau :

1. Il reçoit des **entrées** (x1, x2, ..., xn)
2. Chaque entrée est multipliée par un **poids** (w1, w2, ..., wn)
3. Les résultats sont **sommés**
4. Une **fonction d'activation** transforme le résultat

```
sortie = activation(w1*x1 + w2*x2 + ... + wn*xn + biais)
```

Le **biais** permet d'ajuster le seuil de déclenchement du neurone.

## Architecture d'un réseau

Un réseau de neurones typique comporte trois types de couches :

### Couche d'entrée (Input Layer)
Reçoit les données brutes. Chaque neurone représente une feature.

### Couches cachées (Hidden Layers)
C'est là où la "magie" opère. Plus il y a de couches cachées, plus le réseau est **profond** (d'où le terme "deep learning").

### Couche de sortie (Output Layer)
Produit le résultat final : une classification, une prédiction, etc.

## Les fonctions d'activation

Les fonctions d'activation introduisent de la **non-linéarité**, ce qui permet au réseau de modéliser des relations complexes.

### ReLU (Rectified Linear Unit)
```
f(x) = max(0, x)
```
La plus utilisée. Simple et efficace. Renvoie 0 pour les valeurs négatives, x pour les positives.

### Sigmoid
```
f(x) = 1 / (1 + e^(-x))
```
Transforme les valeurs en probabilités entre 0 et 1. Utilisée en sortie pour la classification binaire.

### Softmax
Généralisation du sigmoid pour la classification multi-classes. Chaque sortie représente la probabilité d'une classe.

## L'apprentissage : la rétropropagation

### Forward Pass
Les données traversent le réseau de l'entrée vers la sortie. Chaque neurone calcule sa sortie.

### Calcul de l'erreur (Loss)
On compare la sortie du réseau avec le résultat attendu via une **fonction de perte** (loss function).

### Backward Pass (Rétropropagation)
L'erreur est propagée en arrière pour ajuster les **poids** de chaque connexion. C'est l'algorithme de **gradient descent** qui détermine comment ajuster les poids.

### Répétition (Epochs)
Ce processus est répété de nombreuses fois (epochs) jusqu'à ce que le réseau atteigne une précision satisfaisante.

## Pourquoi "deep" learning ?

Un réseau est dit **profond** quand il contient **plusieurs couches cachées** (généralement plus de 3). La profondeur permet de :

- Capturer des features de plus en plus abstraites
- Modéliser des relations très complexes
- Traiter des données non structurées (images, texte, audio)

**Couche 1 :** Détecte des bords et contours (pour une image)
**Couche 2 :** Combine les bords en formes
**Couche 3 :** Combine les formes en objets
**Couche 4 :** Reconnaît des objets complexes

## Points clés à retenir

- Un réseau de neurones est composé de couches de neurones interconnectés
- Chaque neurone applique des poids, un biais et une fonction d'activation
- La rétropropagation permet au réseau d'apprendre en ajustant ses poids
- Le deep learning = réseaux avec beaucoup de couches cachées
