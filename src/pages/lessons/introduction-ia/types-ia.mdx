## Classification de l'intelligence artificielle

L'IA peut être classée de plusieurs manières. La classification la plus courante distingue les types d'IA selon leur **niveau de capacité**.

## Par niveau de capacité

### IA faible (ou étroite) — ANI

L'**IA faible** (Artificial Narrow Intelligence) est conçue pour accomplir une tâche spécifique. C'est le seul type d'IA qui existe aujourd'hui.

Exemples concrets :
- **Siri / Alexa** : assistants vocaux
- **Recommandations Netflix** : suggestion de contenu
- **Filtres anti-spam** : classification d'emails
- **ChatGPT** : génération de texte (malgré ses capacités impressionnantes, il reste une IA étroite)

### IA forte (ou générale) — AGI

L'**IA forte** (Artificial General Intelligence) serait capable de comprendre, apprendre et appliquer son intelligence à n'importe quel problème, comme un humain.

Caractéristiques attendues :
- Raisonnement abstrait
- Transfert de connaissances entre domaines
- Compréhension du contexte et du sens commun
- Conscience de soi (débattu)

> L'AGI n'existe pas encore. Les estimations sur son arrivée varient de 10 à 100 ans selon les experts.

### Super-intelligence — ASI

L'**ASI** (Artificial Super Intelligence) dépasserait l'intelligence humaine dans tous les domaines. C'est un concept théorique qui soulève des questions éthiques majeures.

## Par approche technique

### Systèmes basés sur des règles

Les **systèmes experts** utilisent des règles définies par des humains ("SI... ALORS..."). Ils sont prévisibles mais limités à leur base de règles.

```
SI température > 38°C ET toux ET fatigue
ALORS suggestion = "Consultez un médecin"
```

### Machine Learning

Le **machine learning** permet aux machines d'apprendre à partir de données sans être explicitement programmées. C'est l'approche dominante aujourd'hui.

Trois sous-catégories :
- **Apprentissage supervisé** : on fournit des exemples étiquetés
- **Apprentissage non supervisé** : la machine trouve des patterns seule
- **Apprentissage par renforcement** : la machine apprend par essais/erreurs

### Deep Learning

Le **deep learning** utilise des réseaux de neurones profonds (nombreuses couches) pour traiter des données complexes comme les images, le son et le texte.

## Applications par domaine

| Domaine | Application | Type d'IA |
|---------|-------------|-----------|
| Santé | Diagnostic médical | Machine Learning |
| Finance | Détection de fraude | Deep Learning |
| Transport | Véhicules autonomes | Deep Learning |
| Commerce | Recommandations | Machine Learning |
| Créativité | Génération d'images | Deep Learning |

## Points clés à retenir

- Toute l'IA actuelle est de l'**IA faible** (étroite), même les modèles les plus avancés
- L'**AGI** reste un objectif de recherche, pas une réalité
- Le **machine learning** et le **deep learning** sont les approches dominantes
- Chaque type d'IA a ses forces et ses limites spécifiques
